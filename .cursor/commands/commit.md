# Commit Changes

Analiza los cambios actuales y crea un commit siguiendo Conventional Commits.

**Tipos permitidos:**
- `feat:` Nueva característica
- `fix:` Corrección de bug
- `style:` Cambios visuales
- `refactor:` Refactorización
- `docs:` Documentación

Ejecuta:
```bash
git add .
git commit -m "mensaje"

Pregunta si quiero hacer push.
