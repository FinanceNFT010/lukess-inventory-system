# Fix Bugs

Analiza el código buscando estos bugs comunes:

1. **Botones invisibles** (text-white faltante)
2. **Errores de delete** (orden de foreign keys)
3. **Categoría no se guarda** (NULL vs empty string)
4. **Stock Total no ordena** (campo calculado)

Para cada bug encontrado:
- Explica el problema
- Muestra la solución
- Aplica el fix

Pregunta antes de hacer cambios destructivos.
