# INSTRUCCIONES ESPECÍFICAS PARA CURSOR AGENT

## Cuando edites archivos visuales (Dashboard, Inventario, etc.)

1. **SIEMPRE verifica contraste de colores**
   - Lee las reglas de `.cursorrules`
   - Usa text-white en gradientes y botones de color

2. **SIEMPRE lee el archivo completo antes de editar**
   - Usa `@file` para dar contexto
   - No edites sin entender la estructura

3. **Para bugs de base de datos**
   - Verifica el orden de operaciones (foreign keys)
   - Consulta el schema en `.cursorrules`
   - Usa transacciones si es necesario

## Proceso de debugging

1. Identifica el archivo exacto con el problema
2. Abre el archivo completo
3. Encuentra la función/componente específico
4. Verifica imports y tipos
5. Aplica el fix
6. Verifica que no hayas roto nada más
